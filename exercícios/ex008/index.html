Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (Não semântica)

Nessa outra frase, temos um termo em destaque usando a tag <strong>( com semântica)

Itálico / Ênfase

Nessa frase, temos um termo em itálico (Não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em> (Semântica)

Texto Marcado

Também podemos marcar um texto dessa forma com a tag <mark>

Texto deletado

Podemos marcar um texto como excluído usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido usando a tag <ins>

Texto sublinhado

Podemos também sublinhar o texto usando a tag <u>

Texto sobrescrito

Podemos inserir coisas do tipo x2+3 usando a tag <sup>

Texto subescrito

Podemos inserir coisas do tipo H2O usando a tag <sub>

Tag de endereço

Existe a tag <adress> para indicar um endereço. Exemplo: Rua Henrique Dias - Varadouro
